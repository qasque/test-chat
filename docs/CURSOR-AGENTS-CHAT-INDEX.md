# Индекс сессий Cursor Agent (заголовки из боковой панели)

Этот файл — **карта тем**, обсуждавшихся в Cursor Agent для этого монорепозитория. Он нужен, чтобы **новая ИИ-сессия или другой разработчик** быстро поняли *о чём шли разговоры*, даже без доступа к локальному `workspaceStorage`.

## Ограничения (важно для ИИ)

- Здесь сохранены **только названия чатов** и **выводы по смыслу**, как они видны в UI (снимок на **2026-04-17**). **Полные транскрипты диалогов в репозиторий не входят.**
- Если нужен дословный контекст — экспортируйте историю отдельно (локальный бэкап Cursor, экспорт/утилиты) и храните **вне** публичного репо, если там были секреты.
- Для архитектуры и путей в коде опирайтесь на **`docs/REPO-CONTEXT.md`** и сами исходники; этот файл — **надстройка к темам**, а не замена README.

## Как использовать в новом чате

1. Прочитать **`docs/REPO-CONTEXT.md`** (структура репо, сервисы, потоки сообщений).
2. По таблице ниже найти **близкую тему** и связать её с каталогами (`bridge/`, `apps/web/`, `ai-bot/`, скрипты Chatwoot и т.д.).
3. Если тема про мобильный клиент — помнить: **Flutter-приложения в других репозиториях** (см. `REPO-CONTEXT.md`).
